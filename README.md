Statistical Business Analysis
📌 Project Overview

This project performs a comprehensive statistical analysis on business sales data. The objective is to apply core statistical techniques used in Data Science to extract insights and support business decision-making.

The analysis includes:

Descriptive Statistics

Distribution Analysis

Correlation Analysis

Hypothesis Testing

Confidence Interval Estimation

Linear Regression Modeling
